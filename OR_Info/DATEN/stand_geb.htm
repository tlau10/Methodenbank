Standortplanung

(noch nicht voll funktionsfähig)

Mit dem Programm „Standortplanung V1.0“ kann eine optimale Gebietsabdeckung durch Eingabe der Entfernung (in km oder min) errechnet werden. Als Ergebnis werden jene Standorte gezeigt, welche für eine optimale Gebietsabdeckung in Frage kommen.

Menü

Datei

Neu
Hier kann eine neue Eingabematrix angelegt werden. Der Default-Wert ist auf 3 Standorte eingestellt.

Öffnen…
Öffnet eine gespeicherte Datei. Dateien, die nicht mit „Standortplanung V1.0“ erstellt wurden, werden nicht akzeptiert.

Speichern
Speichert die vorhandene Datei. Wurde noch kein Dateiname gewählt, erscheint ein entsprechender Dialog.

Speichern unter…
Hier kann die Datei in einem gewünschten Verzeichnis gespeichert werden.

Eingabe

Zunächst startet das Programm mit dem Default-Wert 3 für die Anzahl der Standorte. Klickt man auf den Matrix Button, wird eine Tabelle erstellt. In diese Tabelle können nun die einzelnen Entfernungen (in km oder min) eingetragen werden. In die Diagonale wird automatisch eine Null geschrieben. In das Feld Maximale Entfernung muss die maximal erlaubte Entfernung für die Berechnung eintragen werden. Hier sollte die gleiche Masseinheit wie in der Tabelle stehen. Zuletzt muss über den Menüpunkt Solver der entsprechende Berechnungsalgorithmus gewählt werden. Über den Button Berechnung wird dann in einem zusätzlichen Fenster das Ergebnis ausgegeben.

CLP und LP-File des Vorlesungsbeispieles Produktionsplanung

CLP-FILE:
Achten Sie beim CLP-File auf den korrekten Eintrag des LP-Filenamens sowie der Ausgabedatei. Das Eingabefile (LP-File) heisst hier MUSTER.LP. 

MUSTER.LP       OUTPUT MUSTER.OUT
                PAGESIZE 24
                LINESIZE 79
                TMARGINS 0
                BMARGINS 0
                FIELDSIZE 11
                DECIMALS 5
                EUROPEAN NO
                LMARGINS 0
                COPIES 1
                WAIT NO
*
                MUTE NO
                LISTINPUT YES
                MARNING YES
                SOLUTION YES
                CONSTRAINTS YES
                COSTANALYSIS YES
                MARGINANALYSIS YES
                MATLIST NO
                DEFAULTS NO

LP-FILE:
Beim LP-File ist darauf zu achten, dass die Sektionen TITLE, OBJECTIVE, BOUNDS und CONSTRAINTS wie hier angegeben enthalten sind. Wird kein Titeltext eingetragen, so muß hierfür zumindest ein Tab-Zeichen gesetzt werden.
Die Optimierungsfunktion lautet hier auf Maximierung. 

..TITLE
        PRODPLAN
..OBJECTIVE MAXIMIZE
        1 x1 + 2 x2
..BOUNDS
        x1 >= 0
        x2 >= 0
..CONSTRAINTS
        3 x1 + 2 x2 <= 12
        1 x1 + 3 x2 <= 9

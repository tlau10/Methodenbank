LP90

***********************************************
Läuft nicht unter Windows NT !
***********************************************

Das LP-Programm beinhaltet eine Benutzerführung. Eine Kurzanleitung zum Programm enthält die Datei DokuLp90.doc. Das Buch Lineare Programmierung auf dem PC von Herrn Prof. P. Stahlknecht enthält neben einer ausführlichen Dokumentation auch den Quellcode.
Es ist in der FH Bibliothek ausleihbar (WCL 6).

Hinweis:
Das Programm wird vom Server in das Verzeichnis c:\temp\lp90 kopiert. Starten müssen Sie das Programm manuell im Explorer. Doppelklick auf LP90.COM.


Das OR-Standardbeispiel in LP90: 

1x1 + 2x2 ---> max
3x1 + 2x2 <= 12
1x1 + 3x2 <=  9

+------------------------------------------------------+
|ALLGEMEINE INFORMATION|NAME             x1       x2   |
|VOM PROGRAMM          |KONSTANT                       |
|                      |UNTERGRENZE                    |
|                      |OBERGRENZE                     |
|                      |ZIEL             1        2    |
+------------------------------------------------------+
|NAME     TYP     RHS1      RHS2    |                  |
|                                   |                  |
|r1       <                  12     |    3        2    |
|r2       <                  9      |    1        3    |
+------------------------------------------------------+


Folgende Parameter sind optional:

Name         Variablennamen
Konstant     Konstantwerte der Variablen
Untergrenze  Untergrenze der Variablen
Obergrenze   Obergrenze der Variablen
RHS1         nur bei zweiseitigen Restriktionen


Folgende Parameter sind Musseingaben:

RHS2         B-Vektor
Typ          Gleichungstyp (<, > und =);
             (":" bei zweiseitigen Restriktionen)
Ziel         Zielfunktionswerte

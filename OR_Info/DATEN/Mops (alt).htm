MOPS

Kurzanleitung für den PC-Client

Akt. Client-Version: 98

MOPS® (Mathematical OPtimization System) ist ein Softwaresystem zur Lösung großer linearer Optimierungsmodelle. Die an der FH eingesetzte Version 2.0 läuft auf dem Unix-Rechner (rz-uxazs) und wird vom PC-Netzwerk aus über das Client-Programm Mops_PC gestartet.

Weitergehende Informationen zu MOPS sind u.a. in der Datei KURZANL.DOC im Verzeichnis ..\BESF\SOLVER\MOPS_98\DOC\ zu finden. Die Datei MOPS_INF.DOC im Verzeichnis ..\BESF\INFO\ enthält zusätzlich eine einführende Beschreibung des MPS-Formats. Das Benutzerhandbuch ist ausleihbar.

Vorgehensweise:

  • MPS-Datei erstellen (MPS ist das Standardformat von LP-SW)
  • MOPS-Client mit Parametern (MPS-Datei, MIN | MAX) aufrufen
  • Ergebnisausgabe in die Datei MOPS.OUT (bzw. OUT.TXT bei Win 3.x)

Start von MOPS unter Win 9x/NT

Der MOPS-Client kann nur noch vom Server gestartet werden.

Aufruf vom Server:
L:\BESF\SOLVER\MOPS_98\EXEC\MOPS_PC.EXE [MPS-Datei] MIN|MAX

Ergebnisdatei: MOPS.OUT

Start von MOPS unter Win 3.x

Aufruf vom lokalen Rechner:
C:\OR\SOLVER\MOPS\CLIENT\MOPS_PC.EXE [MPS-Datei] MIN|MAX

Ergebnisdatei: OUT.TXT

Hinweis:
Seit dem Versionswechsel des Unix-Betriebssystems HP-UX von 9.x auf 10.20 funktioniert der Mops-Aufruf nur noch sporadisch.

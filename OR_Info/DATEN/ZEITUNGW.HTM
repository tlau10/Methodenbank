Zeitungsverkäuferproblem

Ein Zeitungsverkäufer kauft Zeitungen für einen Kaufpreis Pk. Die Zeitungen werden für einen Verkaufspreis Pv wiederverkauft. Die nicht verkauften Zeitungen können für einen bestimmten Rückgabepreis Pr an dem Verlag zurückgegeben werden.

Für die Menge der zu bestellenden Zeitungen existiert eine gewisse Verteilung. Diese Verteilung gibt die Wahrscheinlichkeit für eine Nachfrage N an.

Es gilt eine Gewinnmaximierung hinsichtlich der gekauften und wiederverkauften Zeitungen zu erzielen.

Die Berechnungsformel (vgl. Mayer/Hansen Seiten 200-204) für die Gewinnmaximierung lautet:

     Pv - Pk
     ------- <= F(q)
     Pv - Pr

Für das Programm bedeutet dies:
Die Verteilung, der Einkaufspreis, der Verkaufspreis und der Rückgabepreis müssen bekannt sein. So kann eine Kumulierung der Wahrscheinlichkeiten vorgenommen werden, bis die Summe größer als der Quotient aus der Division ist.

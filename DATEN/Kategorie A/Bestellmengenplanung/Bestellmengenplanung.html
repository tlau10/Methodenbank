Bestellmengenplanung

Beschreibung der Methode:

Diese Methode ermöglicht es, eine optimale Lösung auf Basis der linearen Programmierung für ein Bestellmengen-Problem zu finden. Das Problem besteht dabei aus einer Menge an Perioden. Jede dieser Perioden umfasst den Bedarf einer speziellen Resource, die Kosten für den Bestellprozess dieser Resource und deren Lagerkosten. Optional können Fehlmengenkosten angegeben werden. Die Lösung des linearen Optimierungsproblems liefert die optimalen Mengeneinheiten der benötigten Resource je Periode zu minimalen Kosten. Dies ist eine Online Methode.

Iterator 2013

Der Iterator 2013 ermöglicht es lineare Modelle mit Hilfe des Gauss'schen Eliminationsalgorithmus zu lösen. Grundlage hierfür ist ein Ausgangstableau in Form eines LP-Ansatzes.

Theoretische und Mathematische Grundlagen siehe Vorlesungsunterlagen - Operations Research, Herr Prof. Dr. Grütz, HTWG Konstanz

Folgende Grundfunktionen stehen zur Verfügung:

  • Pivot
    Ermittelt anhand der Pivotspalten- und Zeilenkriterien die zu pivotierende Zelle. Diese wird über Auswahl der entsprechenden Zelle visualisiert.

  • Iteration
    Führt das Gauss'sche Eliminationsverfahren für die gewählte Zelle aus.

  • Optimierung
    Wiederholt führt die Iteration über gefundene Pivotelemente solange aus, bis kein Pivotelement mehr gefunden werden kann.

Weitere Programmfunktionen:

  • Sprache
    Das Programm ist internationalisiert. Die Sprache kann über "Settings" (Einstellungen)-> "gewünschte Sprache" ausgewählt werden.

  • Debug
    Über "Help (Hilfe)"-> "Debug" wird Einblick in die internen Programmabläufe gewährt. Dies ist als Stütze bei der Fehlerbehebung o.ä. dienen.

  • Persistierung
    Durch die Standardfunktionen (Save, Load) kann das gewählte Tableau persistiert werden. Als Datentyp für die Speicherung des Tableaus wird XML verwendet. Neben der erwähnten XML-Formate können auch Dateien im ORI-Format (Iterator 1996) geladen werden.

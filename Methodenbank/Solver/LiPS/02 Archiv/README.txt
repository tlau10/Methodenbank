Linear Program Solver (LiPS) is intended for solving linear programming models. LiPS has easy to use graphical interface. Main features: informative solving reports, extended sensitivity analysis, mixed integer models engine and more. 
LiPS supports MPS format and simple LP format (like lpsolve).

WS 96/97
--------

Neues Solverkonzept: eine .pif und eine .bat. Die Batch-Datei
wird bei jedem Lauf neu erstellt. Ihr Inhalt ist abhängig vom
gewählten Solver. Die Batchdatei ermöglicht den Start der Solver
auf den schreibgeschützten Verzeichnissen des Servers.

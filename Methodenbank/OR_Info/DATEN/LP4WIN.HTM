LP4WIN

LP4Win solves linear programming problems containing 63 variables and 63 constraints using the simplex method. Constraints may be less than, greater or equal to. Z function may be maximized or minimized. Program displays and prints: user input, computed answers, initial and final tableau. Program has special STEP function which shows how tableau is constructed and solved step by step. Spreadsheet like input screen allows for easy input and display of entered data.
Many special features and options for data formatting, such as zero blanking. [F1 key] HELP system.

Written in Visual Basic 2.0 (requires MS Windows and VBRUN200.DLL).

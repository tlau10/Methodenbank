LiPS - Solver

Beschreibung der Methode:
Der Linear Program Solver (LiPS) dient zur Lösung von linearen Programmierproblemen. Die Software zeichnet sich durch eine grafische Oberfläche, sowie eine integrierte Sensibilitätsanalyse aus. LiPS unterstützt die Dateiformate MPS und ein einfaches LP-Format.

Bei dieser Methode handelt es sich um eine externe Software.

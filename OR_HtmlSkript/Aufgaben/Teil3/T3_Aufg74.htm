Aufgabe 74

[SS 2002 A7]

Gegeben sei folgendes Standardbeispiel eines mathematischen Modell:

                 x1  +  2 x2 ---> max !

               3 x1  +  2 x2  <= 12;
 
                 x1  +  3 x2  <= 9;

                 x1, x2  >= 0,   ganzzahlig!

Entwickeln Sie für dieses Beispiel ein spezielles Lösungsverfahren, das über sämtliche Lösungsalternativen vollständig enumeriert (nicht Dakin oder Gomory!). Geben Sie die Schritte des von Ihnen entwickelten Verfahrens an und markieren Sie die Optimallösung!

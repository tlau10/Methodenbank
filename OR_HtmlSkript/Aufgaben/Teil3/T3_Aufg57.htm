Aufgabe 57

[WS 2000 A4]

Zur Produktionsprogrammplanung sei ein LP-Modell der folgenden Standardform gegeben.

    k * x1 + p * x2 → max !

    A 

    *

    x1
    x2

    ≤

    b1
    b2
    b3
    b4
    x1, x2 ≥ 0


Im Laufe der Berechnung mit dem Simplexalgorithmus erhält man nachstehendes Tableau.

       x1     x2     x3     x4     x5     x6    |  R.H.S
--------------------------------------------|----------
   1      0     5/9     0      0    -1/3    |    4
   0      0    -8/9     1      0     1/3    |    9
   0      0     1/9     0      1    -2/3    |    2
   0      1    -2/9     0      0     1/3    |    3 
--------------------------------------------|----------
   0      0     5/9     0      0     2/3    |    Z

  1. Handelt es sich bei dem angegebenen Tableau um das Optimaltableau ?
    Begründen Sie Ihre Antwort.

  2. Berechnen Sie k, p und Z.

  3. Sind alle vorhandenen Kapazitäten voll ausgeschöpft ?
    Begründen Sie Ihre Antwort und geben Sie gegebenenfalls die freien Kapazitäten an.

  4. Welche Aussage uber den Zielfunktionswert können Sie treffen, wenn

    1. b2 um eine Einheit erhöht wird ?
    2. b4 um eine Einheit erhöht wird ?
    3. b4 um fünf Einheiten erhöht wird ?

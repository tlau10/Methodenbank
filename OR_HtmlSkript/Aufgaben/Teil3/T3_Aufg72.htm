[SS 2002 A4]

Gegeben sei folgendes mathematisches Programm:

      f(x1) + 3/4 x2 ---> max!

      x1 + 3/2 x2 <= 6;

      3x1 +  2x2  <= 9;      x1, x2 >= 0;

wobei f(x1) folgendermaßen definiert ist:

                 1/4 x1          für 0 <= x1 < 1

      f(x1)  =   3/4 x1 - 1/2    für 1 <= x1 < 2 

                 2x1 - 3         für 2 <= x1 <= 3

Überführen Sie obiges Programm in einen Ansatz der Linearen Programmierung und lösen Sie es mit dem Solver "Iterator", bitte geben Sie Ausgangs- und Optimaltableau an!

Welches Ergebnis liefert der "Weidenauer-Solver"?

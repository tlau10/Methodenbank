Aufgabe 8

[WS 89/90 A8]

Zum primalen LP-Modell in Standardform: 

       cTx --> max
       Ax  ≤  b
       x   ≥  0

gehört das duale Modell: 

       uTb --> min
       uTA  ≥  cT
       uTx  ≥  0T


Beweisen Sie folgende Behauptung

Wenn sowohl das primale als auch das duale Modell zulässige Lösungen besitzen, so gilt für diese: 

       cTx ≤ uTb

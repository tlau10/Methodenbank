Perl 5 for Win32

Perl is a language that combines some of the features of C, sed, awk and shell. See the Perl 5 UN*X Distribution manual pages for more hype. Note that not all of the functionality in these pages is supported by Perl 5 for Win32.
There are also two Nutshell Handbooks published by O'Reilly & Assoc. for Perl that are a good reference. See the "README.TXT" for more sources of information. Included in the Perl 5 for Win32 Manual is the following:
  • The Perl 5 Manual
  • Unsupported functions and features
  • OLE Extensions
  • Win32 Extensions
  • Sources of more information

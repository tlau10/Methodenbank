Das Variantenfenster

Es steht nun eine Tabelle zur Verfügung, die maximal 100 verschiedene Varianten aufnehmen kann.

Die Spalte "Verschnitt" muss nicht eingegeben werden, das Programm berechnet den Verschnitt für die eingegebene Variante automatisch.

Hinweis:Taucht in der Spalte "Verschnitt" ein negativer Wert auf, so ist die eingegebene Variante zu groß für das gewählte Ausgangsmaterial. Diese Variante muss geändert werden, sonst kann die Berechnung nicht gestartet werden.

Eindimensionale Verschnittproblematik

Vorgegeben sind Rollen in einer bestimmten Breite. Diese Rollen sollen nun in Rollen vorgeschriebener neuen Breiten "geschnitten" werden und zwar längsseits, d.h. die Rollenlänge bleibt immer gleich, und außerdem so, dass dabei ein Minimum an Verschnitt ("Abfall") entsteht. Dabei kann die gewünschte Rollenanzahl pro Breite auch unterschiedlich sein.

Zur Veranschaulichung seien z.B. Rollen mit einer Breite von 5 Metern vorgegeben, die in drei Rollen à zwei Meter und in zwei Rollen à 1,5 Meter "geschnitten" werden sollen. Dieses Beispiel ist trivial:

Es werden zwei zu "schneidende" Rollen benötigt und als Verschnitt ist eine Rolle mit einer Breite von einem Meter anzugeben.

Hier kann der Benutzer vorerst auch die maximale vorhandenen Werkstücke begrenzen, um den verschnitt für diese angegebenen Anzahl von Werkstücke zu optimieren.

Wie kann das eindimensionale Verschnittproblem nun allgemein gelöst werden?

Auch hier wird immer ein lineares Modell formuliert werden, das mit Hilfe der Simplex-Methode gelöst werden kann.

Über dieses Programm

Die Verschnittproblematik ist in Visual Basic 6.0 geschrieben und läuft unter MS Windows ab Version 3.x.

Folgende Systemdateien müssen im Verzeichnis \Windows\System vorhanden sein (auch unter Win95!): cmdialog.vbx, grid.ocx, msole2.vbx.

Der Programmaufruf lautet "VERSCHN.EXE"; bezüglich der zugehörigen Dateien und Verzeichnisstrukturen sind die vorgegebenen Richtlinien laut "RICHTLIN.WRI" strikt eingehalten worden.

Diese Hilfedatei wurde mit dem Tool "MS Help Workshop" von Microsoft Visualstudio 6.0 Software erstellt.

Es folgen noch bekannte Fehler und Verbesserungsvorschläge.

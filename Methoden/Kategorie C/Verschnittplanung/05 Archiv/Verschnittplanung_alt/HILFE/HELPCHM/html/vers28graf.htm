Grafische Ausgabe

Die grafische Ausgabe soll dabei helfen schnell zu erkennen welche Variante den meisten Verschnitt verursacht. Sie kann über den Button "Graphische Darstellung" nach erfolgreicher Berechnung erreicht werden.


Es handelt sich hierbei um ein Balkendiagramm. Ein Balken ist in zwei Farben aufgeteilt, wobei die Farbe "rot" den Verschnitt bei einer Variante darstellt. Die Ansicht der Grafik kann in 2D oder 3D erfolgen.

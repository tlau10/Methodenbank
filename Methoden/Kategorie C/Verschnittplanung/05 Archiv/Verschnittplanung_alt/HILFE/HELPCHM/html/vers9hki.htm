Das Eingabefenster

Zunächst muss ausgewählt werden, welche Art von Problem behandelt werden soll. Hier steht die Auswahl zwischen Verschnittproblem und Werkstückoptimierung. Des weiteren stehen zu jedem zu behandelnden Problem das eindimensionale oder das zweidimensionale Verschnittproblem bzw. eindimensionale oder das zweidimensionale Werkstückoptimierung eine Radiobutton zur Verfügung.

1.1. Das eindimensionale Verschnittproblem

Bei der Berechnung von Verschnittprobleme soll zunächst die Auswahlbutton (Radio Button ) auf Verschnitt zeigen.

Dann ist die Breite des Ausgangsmaterials (der vorgegebenen Rolle) anzugeben. Es sei an dieser Stelle noch einmal erwähnt, dass grundsätzlich immer nur das Eingeben von positiven "Real"-Zahlen erlaubt ist, jede andere Eingabe wird nicht angenommen.

Im folgenden ist nun eine Tabelle zu füllen, und zwar ist je gewünschter neuer Rolle jeweils ihre Breite und ihre Anzahl (wieviele Rollen dieser Breite sollen hergestellt werden?) anzugeben. Es können bis zu 25 verschiedene Rollen (unterschiedlicher Breite) angegeben und innerhalb eines Auftrags (Problems) "geschnitten" werden. Da nicht alle 25 möglichen Rollenbreiten gleichzeitig auf dem Bildschirm angezeigt werden können, kann mittels einer Laufleiste vor- und zurückgesehen werden. Beim Füllen der Tabelle läuft die Tabelle automatisch weiter, wenn über den momentan sichtbaren Bereich hinaus geschrieben wird.

Zusätzlich kann man hier in dem Feld „Max. vorhandenen Werkstücke“ die Anzahl der Werkstücke angeben, um den verschnitt genau für diesen Angabe zu berechnen. Ist dieses Feld leer, so wird die zugehörige Restriktion in die Berechnung nicht berücksichtigt und somit nur die minimale Verschnitt berechnet.

Sind alle Eingabefelder gefüllt worden (nicht benötigte Felder bleiben auf Null und werden vom Programm dann nicht berücksichtigt), kann durch Anklicken des Feldes "Berechnung starten" die Lösung des Problems herbeigeführt werden. Zuvor ist es zumeist sinnvoll, die eingegebenen Daten zu speichern (über die Menüleiste, wie von anderen Windows - Programmen längst bekannt).

Bevor "Berechnung starten" erfolgt, sollte sichergestellt sein, dass die richtigen Verzeichnisse für die Solver eingestellt sind.

1.2. Das zweidimensionale Verschnittproblem

Zusätzlich zur Breite muss auch die Höhe des Ausgangsmaterials eingegeben werden.

Dasselbe gilt für die gewünschten Werkstücke.

Wie bekannt gibt es beim zweidimensionalen Zuschnitt keine Möglichkeit, die Varianten berechnen zu lassen. Deshalb müssen diese nach Betätigen des Buttons "Varianten eingeben" von Hand bestimmt und eingegeben werden.

2.1. Das eindimensionale Werkstoffoptimierung

Bei der Berechnung von Werksoffoptimierung soll zunächst die Auswahlbutton (Radio Button ) auf Werkstück zeigen.

Da die Eingabe von einer Anzahl von Werkstücke bei dieser Art der Berechnung nicht notwendig ist, wird hier das Feld „Max. vorhandenen Werkstücke“ ausgeblendet.

Dann ist die Breite des Ausgangsmaterials (der vorgegebenen Rolle) anzugeben. Es sei an dieser Stelle noch einmal erwähnt, dass grundsätzlich immer nur das Eingeben von positiven "Real"-Zahlen erlaubt ist, jede andere Eingabe wird nicht angenommen.

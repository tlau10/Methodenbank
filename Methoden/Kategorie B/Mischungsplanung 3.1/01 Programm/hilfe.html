Dieses Methodenbankprogramm dient zur Berechnung von Mischungsverhältnissen.
Durch das eingeben einer gewünschten Mischung wird eine Anforderung definiert. Anschließend können beliebig
viele Sorten beschrieben werden, mit deren Hilfe diese Mischung zusammengestellt werden soll.
Das Programm ist an einen Solver angeschlossen, der dann ein abgeleitetes LP-Modell löst und das entsprechende Ergebnis angezeigt.

Beispiel für eine Anwendung wäre eine bestimmte Metalllegierung, die aus verschiedenen Erzen zusammengestellt werden soll.
Eine andere Möglichkeit wäre eine Teemischung, die aus bestimmten Kräutern hergestellt werden soll.

GUI





--> 1 - Fügt der Anforderungsliste eine Anforderung hinzu.

--> 2 - Löscht die selektierte Anforderung aus der Anforderungsliste.

--> 3 - Fügt der Sortenliste eine Sorte hinzu.

--> 4 - Löscht die selektierte Sorte aus der Sortenliste.

--> 5 - Startet die Berechnung.Anschliessend werden Ergebnisse ausgegeben (Nr.11/12).

--> 6 - Schaltet die Ganzzahligkeit für den LP-Solve ein.

--> 7 - Anforderungsliste.

--> 8 - Sortenliste.

--> 9 - Spaltenwert wird automatisch von den Anforderungen übernommen

--> 10 - In dieser Spalte werden die Werte der Sorte eingegeben

--> 11 - Ausgabe der Ergebnisse, wieviel von der jeweiligen Sorte benötigt werden<

--> 12 - Ausgabe der Ergebnisse, wie hoch die Kosten sind

LP-Ansatz



Zielfunktion: 2.5 x1 + 3.0 x2 + 1.7 x3 --> min
Restriktion 1: 2.0 x1 + 3.5 x2 + 1.5 x3 >= 5.0
Restriktion 2: 1.5 x1 + 0.0 x2 + 0.3 x3 >= 2.0
Restriktion 3: 0.2 x1 + 0.0 x2 + 1.0 x3 >= 4.5
Restriktion 4: 0.1 x1 + 2.0 x2 + 0.1 x3 >= 0.5
Zu jeder Berechnung werden dem Nutzer folgende Dateien im Arbeitsverzeichnis erstellt:

  • lpsolve_batchOutStream.txt

  • lpsolve_batchErrorStream.txt --> Enthält Informationen, falls Fehler bei der Berechnung aufgetreten sind

  • lp_ein.lp --> Enthält den LP-Ansatz

  • lp_aus.txt -->Enthält die Lösung der Berechnung

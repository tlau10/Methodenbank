Dieses Methodenbankprogramm dient zur Berechnung von Mischungsverhältnissen. Nachdem eine Anforderung definiert wurde, das heißt eine gewünschte Mischung eingegeben wurde, können eine beliebige Anzahl Sorten beschrieben werden, mit deren Hilfe diese Mischung zusammengestellt werden soll. Das Programm ist an mehrere Solver angeschlossen, die dann ein abgeleitetes LP-Modell lösen. Daraufhin wird das Ergebnis angezeigt.

Beispiel für eine Anwendung wäre eine bestimmte Metalllegierung, die aus verschiedenen Erzen zusammengestellt werden soll. Eine andere Möglichkeit wäre eine Teemischung, die aus bestimmten Kräutern hergestellt werden soll.

Bedienung


In diesem Abschnitt finden Sie eine Erklärung der Bedienung des Programms.

Menü

 

Datei


--> Neu - Legt ein neues Formular für die Mischungsoptimierung an.

--> Öffnen - Öffnet ein gespeichertes Formular (XML-Format).

--> Speichern - Speichert die Eingaben in einem XML-File.

--> Beenden - Beendet das Programm.

Solver


--> LP - Steuert LP-Solve an, um das Modell zu lösen.

--> XA - Steuert XA an, um das Modell zu lösen.

--> Pfade... - Setzen der Solverpfade.

?


--> Hilfe - Startet die Online-Hilfe.

--> Info - Öffnet den Info-Dialog.

GUI



--> 1 - Fügt der Anforderungsliste eine Anforderung hinzu.

--> 2 - Löscht die selektierte Anforderung aus der Anforderungsliste.

--> 3 - Fügt der Sortenliste eine Sorte hinzu.

--> 4 - Löscht die selektierte Sorte aus der Sortenliste.

--> 5 - Ruft den Solver auf und berechnet die Lösung.

--> 6 - Schaltet die Ganzzahligkeit für den LP-Solve ein.

--> 7 - Anforderungsliste.

--> 8 - Sortenliste.

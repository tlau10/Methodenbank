Paket Transport Simulation System

© 2016

by Inan Ödünc, Koray Aras


Das Programm „Paket Transport System Simulation ist ein Werkzeug für die Modellierung und Simulation von paketorientieren Netwerken mit integriertem optimierendem Routenplaner. Es können z.B. Routen und Kosten einer beliebigen Streke errechnet werden.

Als Ergebnis werden jene Strecken gezeigt, welche für eine optimale Route in Frage kommen.

Menü

  • Datei

    • Neu

      • Hier kann eine neue Karte bzw. Route geöffnet werden.

    • Modell laden

      • Öffnet gespeicherte Routen, die in einer Xml-Datei gespeichert worden sind. Andere Dateien können nicht geöffnet werden.

    • Modell Speichern

      • Speichert die vorhandene Datei. Sie wird automatisch in einer Xml Datei abgespeichert.

    • Beenden

      • Beendet das Programm „Paket Transport Simulation System“.


  • Ansicht

    In diesem Programm hat der Benutzer zwei Möglichkeiten der Ansicht. Je nach Wunsch gibt es folgende zwei Möglichkeiten.

    • Modell

      • Unter dieser Ansicht, kann der User seine Routen mit den Knoten auf der Karte Karte sehen.

    • Simulation

      • Unter der Simulations Ansicht kann der Benutzer die Strecke mi.

  • Hilfe

    • Hilfe

      • Aufruf dieser Programmhilfe "Paket Transport System Simulation".

    • About

      • Zeigt Versionsnummer, Versionsjahr und Programmentwickler an.

Buttons

  •  Analog zum Menüpunkt "Datei-Neu" wird eine Karte erstellt.
  •  Analog zum Menüpunkt "Modell laden" wird eine gespeicherte Datei im xml.Format geöffnet.
  •  Analog zu den Menüpunkten "Modell speichern" wird eine vorhandene Datei gespeichert.
  •  Analog zum Menüpunkt "Ansicht" kann hier die Ansicht "Modell" ausgewählt werden.
  •  Analog zum Menüpunkt "Ansicht" kann hier die Ansicht "Simulation" ausgewählt werden.
  •  Startet die Berechnung alle Route (Es werden alle möglichen routen berechnet vom Start-Knoten zum Ziel-Knoten)
  •  Startet die Berechnung einer Route

Eingabe


    Das Programm startet. Mit Klick auf die "Neu"-Funktionalität (über Menü oder Button) kann eine neue Karte oder Route geöffnet werden.
    In der aktuellen Sicht haben wir die gespeicherte Karte zu sehen. Alternativ besteht natürlich die Möglichkeit, ein bestehendes Modell bzw eine andere Karte zu laden.
    (Auf der Karte kann zunächst die gewünschte Route mit Knoten abgebildet werden.
    Es muss anschließend ein Star- und Zielknoten editiert werden. Mit dem Button "berech. alle Route" werden alle Möglichen Routen zum Ziel ausgerechnet. Falls man nur eine Route ausrechnen lassen möchte, wählt man den Button "berech. eine Route" aus. Als Beispiel hierfür haben wir die Route in der Abbildung oben mit dem Startknoten 0 und dem Zielknoten Hellblau.
    Es besteht die Möglichkeit, die Beschriftungen Aus- bzw Anzuschalten. Anschließend kann das Ergebnis mit den Kosten ausgegeben werden. Zuletzt muss über den Menüpunkt "Datei" das Modell abgespeichert oder die Methode beendet werden.

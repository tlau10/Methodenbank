Installationshinweise für das Projekt
"Paket Transport Simulation System" im Rahmen der Veranstaltung "ABSF"

Installation:

Verzeichnisstruktur:

./bin 
Hier müssen die Batch Sover liegen.
./daten
Hier werden die Bild & Modelldaten abgespeichert.
./temp
Hier werden die temporären Datein erzeugt, die für die Berechnung der Routen benötigt werden.
!!! Achtung !!! Es müssen Schreibrechte vorliegen!

Programmstart:
Das Java Program kann entweder

-direkt
mit der Datei PTSSW.exe zu starten
-indirekt
mit START.BAT

gestartet werden. Die hängt von der jeweiligen VM von JaVa ab.


Projekt im JBuilder:

Die Quellen des Projekts liegen im "Source" Ordner.
Das Projekt kann mit der Projektdatei PTSS.jpx mit dem JBuilder gestartet werden.


benötigte Bibliotheken:

-JSX
ist im Verzeichnis "lib" zu finden und muss u.U. im JBuilder als Biblothek bekannt gemacht werden.

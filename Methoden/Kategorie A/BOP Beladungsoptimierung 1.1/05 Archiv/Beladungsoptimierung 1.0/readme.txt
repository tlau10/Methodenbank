BOP - BeladungsOptimierungsProgramm, v. 1.0

Dieses File beschreibt die Ordnerstruktur dieser Anwendung.

/root:

Datei ABSF1.jpx:	Die Projektdatei für BORLAND JBUILDER7
Datei BOP.exe:	Lauffähiges EXE-File der Anwendung. Kann ohne BORLAND-Umgebung 		zur Ausführung gebracht werden.
BOP.jar:		Anwendung als Java Archiv-Datei
readme.txt:	Diese Datei.
solver.bat:		Batch-Datei für den Aufruf des Solvers

/bsb:
Enthält zwei Beispieldateien, welche in der laufenden Anwendung geladen und berechnet werden können.

/classes:

Enthält die .class Dateien der Anwendung

/doc:

-Enthält die Dokumentation der Anwendung im .doc-Format
-Enthält ausserdem die Dokumentation aller Klassen und Methoden als HTML-File, automatisch erzeugt mit JAVADOC. Der Aufruf erfolgt über die Datei "index.html".

/Solver:
Enthält die Solverdatei "LP_SOLVE.exe".

/src:

Enthält die Sourcedateien im Format ".java".

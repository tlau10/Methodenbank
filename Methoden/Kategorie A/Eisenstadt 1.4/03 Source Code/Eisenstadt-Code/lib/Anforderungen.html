2. Installationshinweise

 

Die JobShop-Anwendung benötigt zwei .DLL-Dateien, die im selben Ordner wie die JobShop.jar liegen. Diese .DLL-Dateien sind jedoch, abhängig vom Betriebssystem für 32, bzw. 64-bit ausgelegt. Es muss daher jeweils die richtige JobShop.jar im richtigen Ordner ausgeführt werden. Es existier eine JobShop.jar für 32-bit und eine JobShop.jar für 64-bit. Ausschlaggebend ist dafür die installierte Java-Version (32-bit oder 64-bit).

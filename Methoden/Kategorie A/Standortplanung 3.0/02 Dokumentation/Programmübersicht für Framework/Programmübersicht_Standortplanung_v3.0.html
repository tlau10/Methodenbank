Standortplanung v3.0

© 2013

by Manuel Thoma, Markus Klemens


Mit dem Programm „Standortplanung v3.0“ kann eine optimale Gebietsabdeckung durch Eingabe der Entfernung (verschiedene Einheiten wie sec, min, km, ...) errechnet werden.

Als Ergebnis werden jene Standorte gezeigt, welche für eine optimale Gebietsabdeckung in Frage kommen.


Zu der Ausführung des Tools wird eine Java Umgebung der Version 7 oder höher benötigt.

Es sollten vor Start der Berechnung auch die Pfade für die Solver "LP Solve" und "MOPS" und das Arbeitsverzeichnis über die Einstellungen überprüft bzw. angepasst werden.

Die Pfade zu den Solvern und dem Arbeitsverzeichnis können dynamisch im Programm angepasst und abgespeichert werden.

Voraussetzung für das Speichern ist hierbei, dass man über die entsprechenden Schreibrechte für das jeweilige Verzeichnis, aus dem die Anwendung gestartet wird, verfügt.

Alternativ kann die Anwendung mit den zugehörigen Ordern in das persönliche Arbeitsverzeichnis kopiert werden.
